================================================================================
  DDJ-FLX4 MASTER TRAINER - HOW TO RUN THE APP
================================================================================

QUICK START (3 COMMANDS):
-------------------------

1. Install dependencies:
   npm install

2. Start the application:
   npm start

3. Browser will automatically open to:
   http://localhost:4200


DETAILED INSTRUCTIONS:
----------------------

STEP 1: Check Prerequisites
----------------------------
Make sure you have Node.js installed:
  node --version   (should show v18 or higher)
  npm --version    (should show 9 or higher)

If not installed:
  - Windows: Download from https://nodejs.org/
  - Ubuntu: sudo apt install nodejs npm


STEP 2: Navigate to Project
----------------------------
Open terminal/command prompt and go to project folder:
  cd /path/to/ddj-flx4-trainer-angular


STEP 3: Install Dependencies
-----------------------------
Run this command (only needed first time or after updates):
  npm install

This may take 2-5 minutes.
You'll see progress bars and package installation messages.


STEP 4: Start Development Server
---------------------------------
Run this command:
  npm start

You'll see:
  - "Building..."
  - "Compiled successfully"
  - "Angular Live Development Server is listening on localhost:4200"

Your browser will automatically open to http://localhost:4200


STEP 5: Use the App!
--------------------
The DDJ-FLX4 Master Trainer will load in your browser.
- Click "Setup Instructions" to connect your controller
- Click "Start Training" to begin Lesson 1
- Complete lessons to earn XP and level up!


TROUBLESHOOTING:
----------------

Problem: "npm command not found"
Solution: Install Node.js from nodejs.org

Problem: "Port 4200 is already in use"
Solution: Run: ng serve --port 4300
          Then open: http://localhost:4300

Problem: npm install fails
Solution:
  1. Delete node_modules folder
  2. Delete package-lock.json file
  3. Run: npm install again

Problem: Browser doesn't auto-open
Solution: Manually open http://localhost:4200

Problem: White screen / errors in browser
Solution:
  1. Stop server (Ctrl+C)
  2. Clear browser cache
  3. Run: npm start again


OTHER USEFUL COMMANDS:
----------------------

Build for production:
  npm run build
  (Output will be in dist/ folder)

Run on different port:
  ng serve --port 4300

Stop the server:
  Press Ctrl+C in terminal
