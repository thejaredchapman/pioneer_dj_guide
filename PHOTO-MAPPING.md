## Photo Creation Workflow

### Recommended Approach:

1. **Take all photos in one session** - Ensures consistent lighting and angle
2. **Use a photo template** - Mark control locations on first photo, replicate
3. **Batch annotate similar photos** - Do all close-ups, then all wide shots
4. **Test incrementally** - Add 3 photos, test, add 3 more, test
5. **Keep originals** - Save unannotated versions for future edits

### Efficient Annotation Process:

```
1. Open photo in annotation tool
2. Draw all arrows first
3. Add all text labels
4. Group arrows + labels for each control
5. Adjust positioning for clarity
6. Export with exact filename
7. Move to next photo
```

## File Size Recommendations

- **Resolution:** 1920x1080 (Full HD) minimum
- **File size:** 200KB - 1MB per photo (JPG quality 80-90%)
- **Aspect ratio:** 16:9 or 4:3 (landscape)
- **Total size:** All 9 photos should be under 10MB combined

## Responsive Display

The app automatically:
- Scales photos to fit screen width
- Maintains aspect ratio
- Centers photos in lesson steps
- Shows caption below each photo
- Falls back to text tags if photo missing
