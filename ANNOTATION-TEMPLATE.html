Step-by-Step: How to Add Annotations

Option 1: Using Photopea (Free, Web-based)

  1. Go to photopea.com
  2. Open your controller photo (File ‚Üí Open)
  3. Select the Line Tool (U key) or Shape Tool
  4. Set stroke to red (#FF0000), width 3-5px
  5. Draw arrows pointing to controls
  6. Add arrowheads: Select line ‚Üí Edit ‚Üí Stroke ‚Üí Add arrow
  7. Select Text Tool (T key)
  8. Add text labels with white color, bold font
  9. Draw black rectangles behind text for readability
  10. Export as JPG (File ‚Üí Export As ‚Üí JPG)

Option 2: Using Preview (macOS)

  1. Open photo in Preview
  2. Click the Markup Toolbar icon (toolbox)
  3. Select the Arrow tool
  4. Draw arrows to controls
  5. Change arrow color to red using the color picker
  6. Select Text tool, add labels
  7. Change text to white and add black border
  8. Save the annotated photo

Option 3: Using Figma (Free, Web-based)

  1. Go to figma.com and sign up
  2. Create new file, import your photo
  3. Use Line tool (L) to draw arrows
  4. Add triangle shapes for arrowheads
  5. Use Text tool (T) for labels
  6. Group labels with background rectangles
  7. Export as PNG or JPG (File ‚Üí Export)
