================================================================================
LESSON CONTENT STRUCTURE
================================================================================

Each lesson in lessons.data.ts contains:

{
  id: 1,                           # Unique lesson number
  title: "Controller Layout",     # Lesson name
  description: "Learn all...",    # Short description
  category: "Beginner Basics",    # Category name
  xp: 100,                        # XP reward
  steps: [                        # Array of lesson steps
    {
      title: "Welcome...",        # Step title
      content: "The DDJ...",      # Markdown content
      tips: ["Take time..."],     # Optional tips
      highlightParts: [...]       # Optional highlights
    }
  ],
  quiz: [                         # Optional quiz
    {
      question: "What does...",   # Question text
      options: ["A", "B"...],     # Answer choices
      correctAnswer: 1,           # Index of correct answer
      explanation: "The..."       # Why it's correct
    }
  ]
}


================================================================================
HOW TO MODIFY
================================================================================

To add a new lesson:
1. Open: src/app/data/lessons.data.ts
2. Copy an existing lesson object
3. Change id, title, description
4. Update category and xp
5. Write your steps array
6. Add quiz questions
7. Save file
8. Lesson appears automatically!

To change colors:
1. Open: src/styles.css
2. Find color definitions
3. Change hex values
4. Save and refresh browser

To add new page:
1. Create component in src/app/pages/
2. Add route to src/app/app.routes.ts
3. Link to it from other pages

   - Upload to hosting provider
