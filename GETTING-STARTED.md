## ğŸ”§ Troubleshooting

### App Issues

**Problem: npm install fails**
- Solution: Update Node.js to latest LTS
- Delete node_modules folder
- Run npm install again

**Problem: Port 4200 in use**
- Solution: \`ng serve --port 4300\`
- Or close other Angular apps

**Problem: Lessons won't unlock**
- Solution: Complete previous lesson
- Check you submitted the quiz
- Look for XP reward in header

### Controller Issues

**Problem: No sound**
- Check USB connection
- Verify audio device in software
- Turn up TRIM and channel faders
- Check master volume

**Problem: Buttons don't work**
- Restart DJ software
- Reconnect controller
- Check controller mapping enabled
- Update software

**Problem: Crackling audio**
- Increase buffer size in settings
- Close other audio apps
- Try different USB port
- Restart computer

For detailed troubleshooting, see Setup Instructions in the app!
